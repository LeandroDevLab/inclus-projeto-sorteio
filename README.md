# 🎲 Projeto Sorteio de Número

Projeto desenvolvido para a **Inclus** — uma iniciativa filantrópica voltada à **acessibilidade e inclusão digital**, com o objetivo de ensinar programação de forma simples, prática e acessível.

---

## 📚 Sobre o Projeto

Este é um projeto introdutório onde o aluno aprende a criar um **sorteador de números** utilizando:

- HTML (estrutura)
- CSS (estilo)
- JavaScript (lógica)

O usuário informa um **valor mínimo** e um **valor máximo**, e o sistema retorna um número aleatório dentro desse intervalo.

---

## 🎯 Objetivos de Aprendizagem

Durante este projeto, o aluno irá aprender:

- Manipulação do DOM (`getElementById`)
- Captura de valores de inputs
- Conversão de dados (`parseInt`)
- Validação de dados
- Estruturas condicionais (`if`)
- Geração de números aleatórios (`Math.random`)
- Organização de projeto em pastas
- Interação com o usuário (eventos de clique)

---

## 🖥️ Como Funciona

1. O usuário digita um número mínimo
2. Digita um número máximo
3. Clica no botão **"Sortear"**
4. O sistema exibe um número aleatório dentro do intervalo informado

---

## ⚠️ Validações Implementadas

O sistema trata erros comuns, como:

- Campos vazios
- Valores inválidos
- Mínimo maior que o máximo

## 🔗 Importante

Certifique-se de que os caminhos dos arquivos estão corretos no HTML:

```html
<link rel="stylesheet" href="./src/css/style.css" />
<script src="./src/js/script.js"></script>
```
